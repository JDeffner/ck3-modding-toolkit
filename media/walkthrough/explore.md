# Take the tour

Open one of your event files and try:

- **Hover** `add_gold` → description and supported scopes
- **Type** inside a `trigger = { }` block → only triggers are suggested; inside `immediate = { }` → only effects
- **F12** on a scripted effect → jumps to its definition (yours or vanilla's)
- **Look right** of a `title =` line → the localized text appears inline; use the lightbulb to edit it without leaving the file
- **F12 in a localization yml** → jumps back to where the key is used in script
- **CK3 Localization: Add Language** → scaffolds a new language from your english files, with english overlaid while you translate

The status bar item (bottom right, "CK3") shows data health at a glance — click it to re-run setup.
